Name:
quicksort

Description:
sort an array in O(nlogn)

InputDescription:
First line of input contains an integer to indicate the number of input arrays;
Following lines are arrays to be sorted, an array per line.

OutputDescription:
Output sorted arrays, an array per line.

InputExample:
1
65 14 6 53 156

OutputExample:
6 14 53 65 156

ExampleExplaination:
Output the input array after sorted.

Counts:
2

TestInput:
68 41 31 53 64 8 322 16 51
8 8 8 8 8 

TestOutput:
8 16 31 41 51 53 64 68 322
8 8 8 8 8
